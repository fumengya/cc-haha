# v0.5.28

> 这一版是 v0.5.27 图片标注与后台任务展示的快速修复：文字标注添加后可以重新拖动位置；自动记忆整理触发的 `dreaming` 任务不再作为聊天流里的普通后台任务卡片出现。

## 修复

- **图片文字标注支持添加后拖动调整位置**。在图片标注弹窗中添加文字后，可以再次点击文字区域并拖动重定位，避免文字落点固定后只能撤销重来的问题。
- **隐藏 auto-dream 的聊天流后台任务卡片**。自动记忆整理仍会在后台运行，但不再在普通对话发送后插入“后台任务 运行中 dreaming”这类容易误解的聊天卡片。
