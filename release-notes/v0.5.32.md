# v0.5.32

> 这一版集中修复上游 v0.4.4 / v0.4.5 合并后暴露的桌面后台任务、Agent 完成续写、停止任务和诊断稳定性问题。

## 重点更新

- **修复 Agent/subagent 完成后主对话不续写**：恢复后台任务 terminal notification 进入主模型输入，避免桌面端已经显示 Agent 完成但主对话没有收到结果。
- **后台任务停止更稳**：桌面端只对仍在运行的后台任务发送 stop 请求；服务端把已完成、已停止、已失败或已不存在任务的 stop 视为幂等成功，避免历史任务点击停止时报错。
- **诊断包版本号正确显示桌面版本**：Electron sidecar 会注入 `APP_VERSION` / `CC_HAHA_DESKTOP_VERSION`，诊断导出的 `app-info.json` 不再回退到 `999.0.0-local`。
- **降低诊断误报噪声**：Windows 下 Agent transcript symlink 因权限失败时继续使用 pointer file fallback，但不再记录为错误；first-party telemetry 的 401/403 上报失败降级为 debug，避免污染 recent errors。
- **任务列表和诊断接口更轻量**：`/api/tasks` 不再重复读取每个 task list；session inspection 的 context-only 请求缩短等待窗口，慢 CLI context 估算会快速回退而不是拖到桌面 API timeout。
