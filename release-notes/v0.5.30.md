# v0.5.30

> 这一版把上游 Claude Code Haha v0.4.4 和 v0.4.5 合并到 Code Council 0.5.x 分支，并保留本地桌面工作台、provider/runtime、compact context 和会话列表定制。

## 重点更新

- **合并上游 v0.4.4 / v0.4.5**：吸收后台任务 drawer、Goal 续跑、provider direct proxy mode、默认会话权限、bypassPermissions 修复、Windows ARM64、浏览器 zoom、Markdown style 注入防护等上游改动。
- **Provider/runtime 更稳**：保留本地 provider alias/context-window 兼容逻辑，同时合入上游 experimental betas、reasoning effort、direct proxy 和首轮 active provider runtime 修复。
- **会话上下文估算保留 compact 语义**：上游将 transcript context estimate 拆成 helper 后，本地继续只统计最新 compact boundary 之后的 live context，避免 compact 后上下文指示器被旧历史固定在接近满载。
- **最近项目和会话列表修复**：修复合并过程中 `listSessions()` 漏带 `filePath` 导致新建 session 被跳过的问题，并验证删除 session 后 recent projects cache 会正确失效。
- **发布和平台 smoke 补齐**：吸收 Windows ARM64、release workflow、package smoke、sidecar/native 相关检查更新，为后续桌面发布做准备。

## 合并说明

- `Sidebar.tsx` 冲突保留本地模块化 Sidebar 实现；上游后台任务感知、长标题 tooltip、force refresh 修复后续可按需手动移植。
- `McpSettings.tsx` 保留本地 tab 化 UI，同时采用上游 MCP secret display redaction。
- `chatStore.ts` 合并后台任务状态、pending tool cleanup、prewarm guard、Dream task lifecycle 与本地 coordinator/solo pipeline replay、message queue drain 逻辑。
- `sessionService.ts` 合并 readJsonl/session summary cache、runtime context metadata、provider context-window 解析和 compact boundary context estimate。
